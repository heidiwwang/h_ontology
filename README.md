# h_onotology - a personal sandbox

## Overview

This is my personal sandbox in which I explore modeling (primarily) policy/legal concepts through ontologies and knowledge graphs. While I can think of a number of applications within a policy/governance context, I primarily work on these for fun in my free time for learning and experimentation purposes. I have made this work public in hopes of building a community around moodeling knowledge and finding some convergence understandings with others, particularly regarding tacit knowledge. 

For any computational models to be meaningful, humans have to be able to interrogate and articulate concepts and ideas that we take for granted for social, economic, and cultural reasons rather than just have them be extremely efficient bias engines. 

## Folder Structure

`/drafts`: Contains the "working copies" of various ontologies and knowledge graphs I am working on over time. These files are subject to frequent changes as I refine my methods and understanding of the concepts.

`/input`: any source files used for scripts.

`/output`: any file produced by the scripts.

`/schemas`: Common schemas/ontologies/taxonomies/vocabularies as .ttl files that my drafts are built on and validated against.

`/scripts`: Various Python scripts I used to support the modeling work.

`requirements.txt`: a list of Python libraries that I have installed in the venv for that helper scripts.

`template.ttl`: duplicate this file to quick start writing graphs by hand.

## Contributions

Contributions are not requested given this is driven a personal interest project.However, I welcome feedback or discussion around Semantic Web modeling. 

Additionally, I called this project 'h_ontology' is because any models I develop could not possibly reflect a singular objective truth. All datasets are reflections of the collector. Therefore, I think it is critical, if you disagree with how I have modelled something, to propose entirely different mapping of the same concepts in an interoperable manner. This would allow us to create a larger collective knowledge base which is reflective of dialectical nature of knowledge.
